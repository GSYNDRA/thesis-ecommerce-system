\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

The rapid growth of online fashion retail has increased the demand for intelligent, responsive, and scalable e-commerce systems. This project proposes the development of an Intelligent Fashion E-Commerce System that integrates an AI-powered chatbot capable of handling real-time customer support and automatically escalating complex queries to human staff through socket-based communication. The chatbot is designed to resolve common inquiries such as order tracking, product details, and return policies, while unresolved issues are redirected to staff who can view the entire conversation history and provide instant assistance.

To ensure stable performance during peak traffic periods, particularly in sale seasons, the system adopts scalable techniques using third-party tools such as Redis, RabbitMQ, or Kafka, allowing the monolithic backend to efficiently manage a large number of concurrent client requests. The proposed system aims to enhance user experience, improve customer support efficiency, and demonstrate an effective method for scaling monolithic architectures in high-demand fashion e-commerce environments. The project will be implemented using modern web technologies, including Node.js, Socket.io, and React.js, and evaluated based on performance, scalability, and user satisfaction.

\textbf{Keywords:} fashion e-commerce, AI chatbot, real-time customer support, scalability, monolithic architecture
